# ひだまり工務店|工務店デモサイト(ポートフォリオ用)

架空の地域密着型工務店「ひだまり工務店」のホームページです。
ポートフォリオ用に制作したデモサイトで、実在の企業・団体とは関係ありません。

## 構成

```
├── index.html   … トップページ(1ページ構成)
├── style.css    … スタイルシート(モバイルファースト)
├── script.js    … ハンバーガーメニュー・絞り込み・フォームなどの挙動
└── images/      … 画像(SVGイラスト)
```

## ページ内容

- **トップ(ヒーロー)** … キャッチコピーと施工事例・無料相談への導線
- **ひだまり工務店の家づくり** … 3つの強み
- **施工事例** … 新築/リフォームで絞り込みできるギャラリー
- **家づくりの流れ** … ご相談からアフターサポートまでの6ステップ
- **会社紹介** … 代表挨拶と会社概要
- **お客様の声** … 3件の声
- **お問い合わせ** … 電話番号とお問い合わせフォーム(デモのため送信されません)

## 主な実装ポイント

- スマホ対応(モバイルファーストのレスポンシブデザイン)
- ハンバーガーメニュー(ESCキー・リンク選択で閉じる、背景スクロール固定)
- スクロールに応じたフェードイン表示(`IntersectionObserver`)
- スマホ表示時の画面下部固定CTA(電話・お問い合わせ)
- フォームの簡易バリデーション(必須チェック・メール形式チェック)
- 依存ライブラリなし(HTML / CSS / Vanilla JS のみ)

## 閲覧方法

`index.html` をブラウザで開くだけで表示できます。
