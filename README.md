**公開URL**: https://amekw310-debug.github.io/-komuten-demo-site/

---

## プロジェクト概要

○○県緑川市を拠点とする地域密着型工務店「株式会社ひだまり工務店」のコーポレートサイト。
注文住宅の設計・施工、リフォーム・リノベーションを主力事業とする。

※ポートフォリオ用に制作した**架空の企業のデモサイト**です。実在の企業・団体とは関係ありません。

| 項目 | 内容 |
| --- | --- |
| 会社名 | 株式会社 ひだまり工務店(架空) |
| 所在地 | ○○県緑川市ひだまり町1-2-3(ダミー) |
| 創業 | 1981年4月 |
| 代表者 | 代表取締役 陽田 守 |
| 従業員数 | 18名(自社大工8名) |
| 事業内容 | 注文住宅の設計・施工、リフォーム・リノベーション、耐震診断・改修 |
| 対応エリア | 緑川市・青葉町・川辺町 ほか近郊(車で30分圏内) |

## サイト構成

1ページ完結型(シングルページ)のサイト。

| セクション | 内容 |
| --- | --- |
| トップ(ヒーロー) | 外観写真+キャッチコピー、施工事例・無料相談への導線 |
| 家づくりの強み | 自然素材・地域密着・自社大工の3つの特長 |
| 施工事例 | 6件のギャラリー(新築/リフォームで絞り込み可) |
| 家づくりの流れ | ご相談〜アフターサポートの6ステップ |
| 会社紹介 | 代表挨拶(写真)+会社概要テーブル |
| お客様の声 | 3件 |
| お知らせ | 日付・カテゴリタグ付きのニュース一覧 |
| 採用情報 | 募集職種3種(大工・現場監督・設計スタッフ) |
| お問い合わせ | 電話番号表示モーダル+フォーム(デモのため送信されません) |
| アクセス | Googleマップ+電車・車での道順、駐車場案内(地図はダミー地点) |

## ファイル構成

```
├── index.html   … トップページ(1ページ構成)
├── style.css    … スタイルシート(モバイルファースト)
├── script.js    … メニュー・絞り込み・モーダルなどの挙動
└── images/      … 画像(写真・SVGイラスト)
```

---

## 技術・実装ポイント

- HTML / CSS / Vanilla JS のみ(フレームワーク・ライブラリ不使用)
- モバイルファーストのレスポンシブデザイン(スマホ・タブレット・PC対応)
- ハンバーガーメニュー(全画面ドロワー、ESCキー・リンク選択で閉じる)
- スマホ表示時の画面下部固定CTA(電話・お問い合わせ)
- 電話番号タップ時は発信前に番号表示モーダルを挟む設計
- スクロールに応じたフェードイン表示(`IntersectionObserver`)
- 施工事例のカテゴリ絞り込み(新築/リフォーム)
- フォームの簡易バリデーション(必須チェック・メール形式チェック)
- 画像はWeb用に最適化(ヒーロー写真 160KB / 代表写真 35KB)
